## プロジェクト名
Jewel-Chase

#### ゲーム概要
- 闇を切り裂く怪盗の逃走劇！
- ジャンル：ランアクション×3Dシューティング
- コンセプト：スピード感あふれるアクションとシューティングの爽快感
- →障害物をジャンプや移動でかわして迫りくる敵を撃ち抜け！

#### 操作方法
- キーボード
- 移動はWASD
- ジャンプはSPACE

- マウス
- 左クリックで射撃

#### 開発環境
- Unreal Engine5.3.2
- C++
- Windows
